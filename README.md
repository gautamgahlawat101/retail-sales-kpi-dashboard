# Retail Sales KPI Dashboard — Power BI Analysis

## Overview

This project presents an interactive KPI dashboard built using Power BI to analyze retail sales performance. The goal is to evaluate key business metrics, identify performance trends, and generate actionable insights for decision-making.

---

## Objective

- Analyze overall sales and profitability  
- Identify key revenue drivers  
- Evaluate category-wise and region-wise performance  
- Provide data-driven business insights  

---

## Data Processing

The dataset was processed using Power Query within Power BI:

- Handled missing and inconsistent values  
- Cleaned and standardized data  
- Transformed columns for analysis  
- Prepared data model for visualization  

---

## Key Performance Indicators

- Total Sales: 12.64M  
- Total Profit: 1.47M  
- Profit Ratio: 11.6%  

---

## Key Insights

- Sales show a consistent upward trend over time  
- Technology category performs best in terms of revenue and profitability  
- Furniture category has low profit margins despite strong sales  
- Revenue is concentrated in major cities such as New York and Los Angeles  
- Some sub-categories generate high sales but low profitability  

---

## Business Recommendations

- Optimize pricing in low-margin categories  
- Reduce excessive discounting  
- Focus on high-performing cities  
- Improve performance of underperforming product segments
